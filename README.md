# Simple Tool-Calling Research Agent

A minimal research agent built with CrewAI, Groq LLM, DuckDuckGo search, and Gradio UI.

## Setup

1. Clone the repo
2. Create virtualenv: `python -m venv venv`
3. Activate it
4. Install deps: `pip install -r requirements.txt`
5. Create `.env` with your Groq key
